# Astrophysics and Cosmology

This repository contains educational and scientific simulations related to
Astrophysics and Cosmology, implemented in Python.

## Objectives
- Explore fundamental concepts of the universe
- Apply physics using computational simulations
- Demonstrate scientific reasoning and coding skills

## Topics Covered
- Hubble's Law and cosmic expansion
- Stellar luminosity
- Observational cosmology (simplified models)

## Disclaimer
This repository is strictly educational and scientific.
No real astronomical data or predictions are claimed.

## Technologies
- Python 3
- Scientific reasoning
- Computational physics
